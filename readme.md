# Prisma Mode Support for Acode

Bring full Prisma syntax support to Acode with a lightweight CodeMirror language mode plugin.

## Features

- Syntax highlighting for Prisma schema files
- `.prisma` file support in Acode
- Built with Lezer grammar for accurate parsing
- Autocomplete-friendly completion snippets
- Easy install and use within the Acode plugin environment

## Installation

1. Open Acode.
2. Go to the Plugins section.
3. Install or add the plugin package.
4. Open a `.prisma` file to enable Prisma mode support.

## Usage

- Rename your Prisma files with the `.prisma` extension.
- Acode will detect the file type and activate the Prisma language mode.
- Enjoy improved syntax highlighting and enhanced editing support.

## About

This plugin is designed to enhance the Prisma development experience inside the Acode editor.
It provides a focused language mode tailored for Prisma schema files without adding unnecessary overhead.
